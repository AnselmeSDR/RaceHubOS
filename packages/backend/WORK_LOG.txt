=== PROTOCOLE CARRERA ===

Format des messages:
- Terminés par '$'
- Checksum: somme des bytes & 0x0F, encodé en ASCII ('0' + checksum)
- Nibbles encodés: valeur 0-15 → '0'-'?' (0x30-0x3F)

Commandes:
- '?' → Poll (demande status/timer)
- 'T' + buttonId + checksum + '$' → Press button
- 'J' + word|addr<<5 + value + repeat + checksum + '$' → Set word
- ':' + mask + checksum + '$' → Ignore controllers
- '=' + 1 + 0 + checksum + '$' → Reset timer

États de course (cuStatus.start):
- 0: Course en cours (GO)
- 1-5: Décompte des feux
- 6: Faux départ
- 7: GO!
- 8+: Course arrêtée

IDs Boutons CU:
- 1: ESC/Pace Car
- 2: START/ENTER
- 5: SPEED
- 6: BRAKE
- 7: FUEL
- 8: CODE

=== PROBLÈMES RÉSOLUS ===

1. Commandes sans effet:
   - Cause: manquait le '$' terminateur
   - Fix: ajouté Buffer.concat([buffer, Buffer.from('$')]) dans controlUnit.js

2. Timeout sur les commandes:
   - Cause: polling interférait avec les réponses
   - Fix: commandes fire-and-forget (pas d'attente de réponse)

3. Arrêter les voitures:
   - setSpeed(0) = limite max, ne stoppe pas instantanément
   - ignore(0xFF) = désactive controllers mais voitures roulent encore
   - Solution: appuyer sur START passe en mode lights (1/5) = voitures stoppées
